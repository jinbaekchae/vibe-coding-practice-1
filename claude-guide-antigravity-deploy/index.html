Python 설치
python.org에서 Python 다운로드
  1. 브라우저에서 python.org/downloads 접속
  2. 페이지 상단의 "Download Python install manager" 버튼 클릭 → .msix 파일 다운로드
    📌 버튼 찾기 팁
    노란 버튼("Download Python 3.x.x")보다 위에 있는 버튼이 Install Manager입니다.
    "Download Python install manager" 또는 "Python install manager 26.x"라고 표시됩니다.
  3. 다운로드된 .msix 파일 더블클릭하여 실행
  4. 설치 팝업에서 "설치" 클릭 → 자동으로 진행됩니다
  5. 완료 메시지가 뜨면 "닫기" 클릭
🖥️ 설치 후 검은 창이 뜨면서 영어로 y/N 을 묻는다면? — 정상입니다
Python Install Manager가 처음 실행되면서 자동으로 "환경 점검"을 합니다. 환경에 따라 영어 질문이 최대 5번까지 뜰 수 있어요. 길어 보이지만 당황하지 마세요.
👉 간단 규칙: 키보드 Y 입력 후 Enter를 그때마다 누르면 됩니다.
(질문이 안 떠도 정상, 5번 다 떠도 정상. 모두 Y로 답해도 안전합니다.)
각 질문이 무엇을 묻는지 (참고용):
영어 질문 무엇을 묻는지 어떻게 답할지
Open Settings now ... ? [y/N] 윈도우 설정 창 열까요? 그냥 Enter
Update setting now? [y/N] 긴 파일 경로 허용할까요? Y + Enter
Open Installed apps now? [y/N] 옛 파이썬 런처 점검 창 열까요? 그냥 Enter
Add commands directory to your PATH now? [y/N] 터미널에서 명령어 인식되게 PATH에 등록할까요? Y + Enter
Install CPython now? [Y/n] ← 핵심! 파이썬 본체(3.xx)를 지금 설치할까요?
이걸 Y해야 실제 파이썬이 깔립니다.		 Y + Enter
마지막에 Configuration checks completed. 메시지가 뜨면 끝. 창은 자동으로 닫힙니다.
💡 설치 후 바탕화면에 아무것도 생기지 않아요 — 정상입니다
Python은 카카오톡이나 크롬처럼 실행하는 앱이 아니라, 컴퓨터 안에 깔리는 언어 엔진입니다. 바탕화면 아이콘이 생기지 않는 게 맞습니다.

Node.js 설치
nodejs.org에서 LTS 버전 다운로드
  1. 브라우저에서 nodejs.org 접속
    ▲ nodejs.org — 초록색 "Get Node.js®" 버튼 클릭
  2. "Get Node.js®" 초록색 버튼 클릭 (LTS 버전이 자동 선택됨)
  3. 다운로드 페이지로 이동되면, 페이지 중간의 코드(명령어)는 무시하고 아래로 스크롤해서 "Windows 설치 프로그램 (.msi)" 버튼 클릭
    ⚠️ 주의 — 코드 블록은 건드리지 마세요
    페이지 가운데에 nvm install 24 같은 코드가 보이는데, 이건 개발자용 방식입니다. 무시하고 페이지 아래 "Windows 설치 프로그램 (.msi)" 버튼을 클릭하세요.
  4. 다운로드된 .msi 파일 실행
  5. 설치 화면에서 계속 "Next" 클릭 (기본값 그대로 OK)
  6. 마지막에 "Install" 클릭 → 관리자 권한 팝업 뜨면 "예"
  7. 완료되면 "Finish" 클릭
📌 Node.js를 설치하면 npm도 자동으로 설치됩니다
npm은 앱스토어처럼 프로그램을 설치하는 도구로, Claude Code 설치에 사용됩니다. 따로 설치할 필요 없습니다.
💡 설치 후 바탕화면에 아무것도 생기지 않아요 — 정상입니다
Node.js도 Python처럼 눈에 보이지 않는 언어 엔진입니다. 바탕화면 아이콘이 생기지 않는 게 맞고, 다음 단계에서 명령어로 사용하게 됩니다.

Git 설치
git-scm.com에서 다운로드 — Git Bash 터미널 포함
  1. 브라우저에서 git-scm.com/download/win 접속
    ▲ git-scm.com — "Click here to download" 링크 클릭
  2. "Click here to download" 링크 클릭하여 다운로드
  3. 다운로드된 파일 실행
  4. 설치 화면에서 계속 "Next" 클릭 (기본값 그대로 OK)
  5. 마지막에 "Install" 클릭
  6. 완료되면 "Finish" 클릭
🖥️
Git 설치 마법사 화면
모든 옵션을 기본값 그대로 두고 "Next" 클릭, 특별히 변경할 항목 없음
📌 Git을 설치하면 "Git Bash"라는 터미널이 함께 설치됩니다
Git Bash는 5단계에서 Claude Code를 설치할 때 사용하는 검은 화면의 터미널입니다. Mac의 기본 터미널과 동일한 명령어를 사용할 수 있어 강의 자료가 동일하게 적용됩니다.

Antigravity 설치
antigravity.google에서 Windows 버전 다운로드
⚠️ 시작 전 꼭 확인 — 로그인 가능한 Gmail 준비
Antigravity는 현재 공개 프리뷰 단계라 개인 Gmail 계정으로만 로그인됩니다. 회사·학교 도메인의 Google Workspace 계정(예: @회사명.com)은 로그인이 거부됩니다. 개인 Gmail 계정(@gmail.com)을 미리 준비해 주세요.
  1. 브라우저에서 antigravity.google/download 접속
    ▲ antigravity.google/download — Windows 칸의 "Download for x64" 클릭
  2. Windows 칸에서 "Download for x64" 버튼 클릭 (대부분 PC가 x64)
    Surface Pro X 같은 ARM 노트북이면 "Download for ARM64"를 선택. 모르겠으면 일단 x64를 누르면 됩니다.
  3. 다운로드된 .exe 파일 더블클릭하여 실행
  4. "Windows의 PC 보호" 파란 경고창이 뜨면 → "추가 정보" 클릭 → "실행" 클릭
    게시자가 "Google LLC" 인지 확인하고 진행하세요. 정상입니다.
  5. 관리자 권한 요청 창이 뜨면 "예"
  6. 설치 마법사 안내에 따라 진행 → 설치 완료까지 대기 (3~5분)
  7. "Finish" 클릭하여 설치 종료 ✅
💡 여기까지가 설치 끝입니다
Antigravity 실행과 Google 로그인은 강의 당일 함께 진행합니다. 설치만 미리 끝내두시면 됩니다.

관리자 권한이 있는 계정으로 로그인하거나, 개인 노트북으로 준비해 주시기 바랍니다.

Git 설치
터미널에서 명령어 한 줄로 설치

Mac에는 Xcode 커맨드라인 도구라는 Apple 공식 패키지가 있는데, 이 안에 Git이 포함되어 있습니다.
아래 명령어 한 줄로 이 패키지를 설치하면 Git이 자동으로 함께 설치됩니다.
  1. 터미널 앱 열기
    방법 1: 키보드 Cmd(⌘) + 스페이스 → "terminal" 입력 → 엔터
    방법 2: 응용 프로그램 → 유틸리티 → 터미널
  2. 아래 명령어를 터미널에 복사 후 붙여넣기 (Cmd+V) → 엔터
    xcode-select --install
    이 명령어는 Apple의 Xcode 커맨드라인 도구를 설치합니다. Git이 이 안에 포함되어 있습니다.
  3. "명령줄 개발자 도구를 설치하시겠습니까?" 팝업이 뜨면 "설치" 클릭
    📌 팝업이 뜨지 않는다면?
    이미 Git이 설치된 것입니다. 다음 단계로 넘어가세요.
  4. 설치 완료(수 분 소요) → "완료" 클릭
4
Antigravity 설치
antigravity.google에서 macOS 버전 다운로드
⚠️ 시작 전 꼭 확인 — 로그인 가능한 Gmail 준비
Antigravity는 현재 공개 프리뷰 단계라 개인 Gmail 계정으로만 로그인됩니다. 회사·학교 도메인의 Google Workspace 계정(예: @회사명.com)은 로그인이 거부됩니다. 개인 Gmail 계정(@gmail.com)을 미리 준비해 주세요.
  1. 브라우저에서 antigravity.google/download 접속
    ▲ antigravity.google/download — MacOS 칸의 "Download for Apple Silicon" 또는 "Download for Intel" 클릭
  2. MacOS 칸에서 내 Mac에 맞는 버튼 클릭:
    • "Download for Apple Silicon" — M1/M2/M3/M4 칩 (2020년 이후 대부분 Mac)
    • "Download for Intel" — 2020년 이전의 인텔 Mac
    모르겠으면 좌측 상단 사과 메뉴 🍎 → "이 Mac에 관하여" → "칩" 항목 확인.
    "Apple M1/M2/..." 라고 적혀 있으면 Apple Silicon, "Intel..." 이라고 적혀 있으면 Intel.
  3. 다운로드된 .dmg 파일 더블클릭
  4. 창이 뜨면 Antigravity 아이콘을 응용 프로그램(Applications) 폴더로 드래그 ✅
💡 여기까지가 설치 끝입니다
응용 프로그램 폴더에 Antigravity 아이콘이 들어갔다면 설치 완료. 첫 실행과 Google 로그인은 강의 당일 함께 진행합니다.
(Mac에서 처음 실행할 때 "확인되지 않은 개발자" 경고가 뜨면, 시스템 설정 → 개인정보 보호 및 보안 → "확인 없이 열기"로 통과 가능)

Claude Code 설치
터미널에서 명령어로 설치
  1. 터미널 앱 열기 (Cmd+스페이스 → "terminal" → 엔터)
  2. 아래 명령어를 터미널에 복사 후 붙여넣기
    npm install -g @anthropic-ai/claude-code
    💡 Mac 터미널 붙여넣기
    복사 후 터미널에서 Cmd(⌘) + V
  3. 엔터 키 누르기
  4. 설치 완료까지 기다리기 — 보통 1~3분, 최대 20분 소요될 수 있음 (텍스트가 흘러가는 것이 정상)
  5. 다시 $ 기호가 나타나면 설치 완료 ✅

Claude Code 설치 진행 화면
명령어 실행 후 텍스트가 흘러가다가 $ 기호가 다시 나타나면 완료

설치 확인
터미널을 새로 열고 4가지 명령어로 확인
⚠️ 반드시 터미널을 새로 열고 확인하세요
기존에 열려있던 터미널 창에서는 방금 설치한 프로그램이 인식되지 않을 수 있습니다. 터미널을 완전히 닫은 후 다시 열어 아래 명령어를 실행해 주세요.

아래 명령어를 하나씩 입력하고 엔터 → 버전 번호가 뜨면 성공입니다.

# 1. Python 확인
python3 --version

# 2. Node.js 확인
node -v

# 3. Git 확인
git --version

# 4. Claude Code 확인
claude --version

정상 출력 예시:

Python 3.14.3
v22.14.0
git version 2.53.0 (Apple Git-137)
1.x.x
💡 버전 숫자가 위 예시와 달라도 정상입니다
예시는 작성 시점 버전이라 시간이 지나면 더 높은 숫자가 표시됩니다. (예: Python 3.15.x, v24.x.x 등) "command not found" 같은 오류 없이 어떤 버전 숫자라도 떴다면 설치 성공입니다.
✅ 4가지 모두 버전이 떴다면 설치 완료!
강의 당일 claude 명령어를 입력하면 아래와 같은 화면이 나옵니다. Claude 계정으로 로그인하면 바로 사용 가능합니다.
🎉 이 화면이 나오면 완전히 설치 완료!
터미널에서 claude 를 입력했을 때 아래처럼 돼지·블록 캐릭터와 함께 "Welcome to Claude Code"가 표시되면 성공입니다.
▲ 이 화면이 나오면 Claude Code 설치 성공! (로그인 URL이 표시됩니다)

문제가 생겼을 때
자주 나오는 오류와 해결 방법
❌ "command not found" 라고 나와요
터미널 창을 닫고 새로 열어 주세요. 새 터미널에서 다시 시도해도 안 된다면 Mac을 재부팅해 보세요.
❌ python3 --version이 오류가 나요
python3 명령을 확인해 주세요 (Mac은 python이 아니라 python3입니다). 설치가 안 됐다면 1단계로 돌아가서 다시 설치해 주세요.
❌ xcode-select --install을 했는데 팝업이 안 뜨고 오류가 나요
이미 설치가 되어 있는 것입니다. git --version을 입력해서 버전이 나오면 정상입니다.
❌ Antigravity 실행 시 "개발자 확인 불가" 경고가 뜹니다
시스템 설정 → 개인 정보 보호 및 보안 → 아래쪽 "확인 없이 열기" 또는 "허용" 버튼을 클릭해 주세요. Apple이 검증한 Google LLC 서명이라 안전합니다.
