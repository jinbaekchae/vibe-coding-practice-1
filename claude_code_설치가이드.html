Python 설치
python.org에서 다운로드 — PATH 설정이 핵심!
⚠️ 가장 중요한 주의사항
설치 화면 맨 아래의 "Use admin privileges when installing py.exe"와 "Add python.exe to PATH" — 두 체크박스를 반드시 모두 체크해야 합니다. 빠뜨리면 이후 Python 명령이 동작하지 않습니다.
  1. 브라우저에서 python.org/downloads 접속
    ▲ python.org/downloads — 노란색 "Download Python 3.x.x" 버튼 클릭
  2. 노란색 "Download Python 3.x.x" 버튼 클릭하여 다운로드
  3. 다운로드된 파일 실행
  4. 설치 화면 첫 페이지 맨 아래에서 아래 두 체크박스를 모두 체크!
    ▲ 두 항목 모두 체크 후 "Install Now" 클릭
    📌 두 체크박스의 역할
    Use admin privileges when installing py.exe — py 런처를 시스템 전체에 설치 (문제 예방)
    Add python.exe to PATH — 터미널에서 python 명령어 사용 가능 (이게 핵심!)
  5. "Install Now" 클릭 → 관리자 권한 팝업 뜨면 "예"
  6. "Setup was successful" 뜨면 "Close" 클릭
📌 이미 설치했는데 PATH 체크를 안 했다면?
제어판 → 프로그램 추가/제거에서 Python을 제거한 후, 위 과정을 다시 진행해 주세요. ("Add Python to PATH" 체크 필수!)
💡 설치 후 바탕화면에 아무것도 생기지 않아요 — 정상입니다
Python은 카카오톡이나 크롬처럼 실행하는 앱이 아니라, 컴퓨터 안에 깔리는 언어 엔진입니다. 바탕화면 아이콘이 생기지 않는 게 맞습니다.

Node.js 설치
nodejs.org에서 LTS 버전 다운로드
  1. 브라우저에서 nodejs.org 접속
    ▲ nodejs.org — 초록색 "Get Node.js®" 버튼 클릭
  2. "Get Node.js®" 초록색 버튼 클릭 (LTS 버전이 자동 선택됨)
  3. 다운로드 페이지로 이동되면, 페이지 중간의 코드(명령어)는 무시하고 아래로 스크롤해서 "Windows 설치 프로그램 (.msi)" 버튼 클릭
    ⚠️ 주의 — 코드 블록은 건드리지 마세요
    페이지 가운데에 nvm install 24 같은 코드가 보이는데, 이건 개발자용 방식입니다. 무시하고 페이지 아래 "Windows 설치 프로그램 (.msi)" 버튼을 클릭하세요.
  4. 다운로드된 .msi 파일 실행
  5. 설치 화면에서 계속 "Next" 클릭 (기본값 그대로 OK)
  6. 마지막에 "Install" 클릭 → 관리자 권한 팝업 뜨면 "예"
  7. 완료되면 "Finish" 클릭
📌 Node.js를 설치하면 npm도 자동으로 설치됩니다
npm은 앱스토어처럼 프로그램을 설치하는 도구로, Claude Code 설치에 사용됩니다. 따로 설치할 필요 없습니다.
💡 설치 후 바탕화면에 아무것도 생기지 않아요 — 정상입니다
Node.js도 Python처럼 눈에 보이지 않는 언어 엔진입니다. 바탕화면 아이콘이 생기지 않는 게 맞고, 다음 단계에서 명령어로 사용하게 됩니다.

Git 설치
git-scm.com에서 다운로드 — Git Bash 터미널 포함
  1. 브라우저에서 git-scm.com/download/win 접속
    ▲ git-scm.com — "Click here to download" 링크 클릭
  2. "Click here to download" 링크 클릭하여 다운로드
  3. 다운로드된 파일 실행
  4. 설치 화면에서 계속 "Next" 클릭 (기본값 그대로 OK)
  5. 마지막에 "Install" 클릭
  6. 완료되면 "Finish" 클릭
🖥️
Git 설치 마법사 화면
모든 옵션을 기본값 그대로 두고 "Next" 클릭, 특별히 변경할 항목 없음
📌 Git을 설치하면 "Git Bash"라는 터미널이 함께 설치됩니다
Git Bash는 5단계에서 Claude Code를 설치할 때 사용하는 검은 화면의 터미널입니다. Mac의 기본 터미널과 동일한 명령어를 사용할 수 있어 강의 자료가 동일하게 적용됩니다.

VS Code 설치
code.visualstudio.com에서 Windows 버전 다운로드
  1. 브라우저에서 code.visualstudio.com 접속
    ▲ code.visualstudio.com — "other platforms" 클릭 후 Windows 버전 선택
  2. 다운로드 버튼 아래의 "other platforms" 링크 클릭
    기본 버튼이 "Download for macOS"로 표시될 수 있습니다. 반드시 "other platforms" → Windows x64 User Installer 선택하세요.
  3. 다운로드된 파일 실행
  4. "사용권 계약 동의함" 체크 → "다음"
  5. 설치 옵션 화면에서 아래 항목 체크 확인:
    • ✅ "Code(으)로 열기" 작업을 파일 상황에 맞는 메뉴에 추가
    • ✅ "Code(으)로 열기" 작업을 디렉터리 상황에 맞는 메뉴에 추가
    • ✅ PATH에 추가 (기본 체크)
  6. "다음" → "설치" → "마침"

  4. 검은 화면의 터미널이 열리면 성공 ✅
🔍
시작 메뉴에서 "Git Bash" 검색

Git Bash에서는 Ctrl+V가 동작하지 않습니다. 터미널 안에서 마우스 오른쪽 클릭 → "Paste" 로 붙여넣기 하세요.

💻 Mac 사용자 안내
Mac은 기본 터미널이 내장되어 있어 Git Bash 없이 바로 사용할 수 있습니다. Windows보다 설치 단계가 적고 PATH 설정 걱정도 거의 없습니다.

Python 설치
python.org에서 macOS 버전 다운로드
  1. 브라우저에서 python.org/downloads 접속
    ▲ python.org/downloads — 노란색 "Download Python 3.x.x" 버튼 클릭
  2. 노란색 "Download Python 3.x.x" 버튼 클릭하여 다운로드
  3. 다운로드된 .pkg 파일 더블클릭
  4. 설치 화면에서 "계속" 클릭 → 사용권 계약 "동의" → "설치"
  5. 비밀번호 입력 (Mac 로그인 비밀번호) → "소프트웨어 설치" 클릭
  6. 설치 완료 후 "닫기" 클릭
📌 Mac은 PATH 체크박스 없어요 — 자동으로 처리됩니다
Windows와 달리 Mac에서는 별도로 PATH를 체크할 필요가 없습니다. 설치 후 자동으로 설정됩니다.

Node.js 설치

    ▲ nodejs.org — 초록색 "Get Node.js®" 버튼 클릭

  3. 다운로드 페이지로 이동되면, 페이지 중간의 코드(명령어)는 무시하고 아래로 스크롤해서 "macOS 설치 프로그램 (.pkg)" 버튼 클릭
    ⚠️ 주의 — 코드 블록은 건드리지 마세요
    페이지 가운데에 코드가 보이는데, 이건 개발자용 방식입니다. 무시하고 페이지 아래 "macOS 설치 프로그램 (.pkg)" 버튼을 클릭하세요.

Git 설치
터미널에서 명령어 한 줄로 설치
💻 Mac에서 Git은 터미널 명령어 한 줄로 설치됩니다
Mac에는 Xcode 커맨드라인 도구라는 Apple 공식 패키지가 있는데, 이 안에 Git이 포함되어 있습니다.
아래 명령어 한 줄로 이 패키지를 설치하면 Git이 자동으로 함께 설치됩니다.
  1. 터미널 앱 열기
    방법 1: 키보드 Cmd(⌘) + 스페이스 → "terminal" 입력 → 엔터
    방법 2: 응용 프로그램 → 유틸리티 → 터미널
  2. 아래 명령어를 터미널에 복사 후 붙여넣기 (Cmd+V) → 엔터
    xcode-select --install
    이 명령어는 Apple의 Xcode 커맨드라인 도구를 설치합니다. Git이 이 안에 포함되어 있습니다.
  3. "명령줄 개발자 도구를 설치하시겠습니까?" 팝업이 뜨면 "설치" 클릭
    📌 팝업이 뜨지 않는다면?
    이미 Git이 설치된 것입니다. 다음 단계로 넘어가세요.
  4. 설치 완료(수 분 소요) → "완료" 클릭

VS Code 설치
code.visualstudio.com에서 macOS 버전 다운로드

Claude Code 설치
터미널에서 명령어로 설치
  1. 터미널 앱 열기 (Cmd+스페이스 → "terminal" → 엔터)
  2. 아래 명령어를 터미널에 복사 후 붙여넣기
    npm install -g @anthropic-ai/claude-code
    💡 Mac 터미널 붙여넣기
    복사 후 터미널에서 Cmd(⌘) + V
  3. 엔터 키 누르기
  4. 설치 완료까지 기다리기 — 보통 1~3분, 최대 20분 소요될 수 있음 (텍스트가 흘러가는 것이 정상)
  5. 다시 $ 기호가 나타나면 설치 완료 ✅

설치 확인
터미널을 새로 열고 4가지 명령어로 확인
⚠️ 반드시 터미널을 새로 열고 확인하세요
기존에 열려있던 터미널 창에서는 방금 설치한 프로그램이 인식되지 않을 수 있습니다. 터미널을 완전히 닫은 후 다시 열어 아래 명령어를 실행해 주세요.

아래 명령어를 하나씩 입력하고 엔터 → 버전 번호가 뜨면 성공입니다.

# 1. Python 확인
python3 --version

# 2. Node.js 확인
node -v

# 3. Git 확인
git --version

# 4. Claude Code 확인
claude --version

정상 출력 예시:

Python 3.14.3
v22.14.0
git version 2.53.0 (Apple Git-137)
1.x.x
✅ 4가지 모두 버전이 떴다면 설치 완료!

문제가 생겼을 때
자주 나오는 오류와 해결 방법
❌ "command not found" 라고 나와요
터미널 창을 닫고 새로 열어 주세요. 새 터미널에서 다시 시도해도 안 된다면 Mac을 재부팅해 보세요.
❌ python3 --version이 오류가 나요
python3 명령을 확인해 주세요 (Mac은 python이 아니라 python3입니다). 설치가 안 됐다면 1단계로 돌아가서 다시 설치해 주세요.
❌ xcode-select --install을 했는데 팝업이 안 뜨고 오류가 나요
이미 설치가 되어 있는 것입니다. git --version을 입력해서 버전이 나오면 정상입니다.
❌ VS Code 실행 시 "개발자 확인 불가" 경고가 뜹니다
시스템 설정 → 개인 정보 보호 및 보안 → 아래쪽 "허용" 버튼을 클릭해 주세요.
